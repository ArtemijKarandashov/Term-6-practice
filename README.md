# Выполненные задачи по практике (6 семестр)

## Задание 1 - ps-map

1. Контейниризовать существующий сервис ps-map при помощи Docker
2. С помощью Cron реализовать автоматичесую сборку проекта через gulp
    - Скрипты для gulp взяты исходные, практически без изменений
    - Cron собирает проект при помощи .sh скрипта (rebuild-ps-map.sh) 1 и 15 числа каждого месяца в 7:00
3. Отправить готовую заархивированную сборку на другую машину
    - После успешной сборки происходит архивация в tar.gz и отправка по ssh с помощью утилиты scp
    - Логи выполнения rebuild-ps-map.sh сохраняются в директории /logs в файле ps-map.log

- Для того чтобы развернуть проект ознакомьтесь с README файлом в директории Task1-ps-map данного репозитория

## Задание 2 - dates

1. Gодготовить dockerfile и docker-compose файлы сервиса к продакшену.
2. Максимально ускорить и оптимизировать процесс установки зависимостей, а также разбить процесс развертывания на несколько этапов.
3. Использовать gunicorn вместо dev сервера django
4. Frontend сервиса реализован на nextjs. Установка зависимостей должна быть реализована через pnpm.
