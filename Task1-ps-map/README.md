# ps-map

## Информация о сервисе

1. Собирает проект с помощью gulp и помещает в директорию dist, архивирует содержимое в build и отправлет с помощью scp файлы на удаленный хостинг.(Путь указывается в .env, переменная HOST_PATH). 
2. Запускается 1 и 15 числа каждого месяца в 7:00.
3. При запуске сборки будут удалены все билды старше 30 дней.

### Изменения исходных файлов
- Изменен файл scripts/gulp/_build.js для работы внутри контейнера.

## Деплой
1. Создать ssh ключ и поместить его в директорию ssh
2. Заменить путь к ключу в .env, подставив туда только что созданый файл (ex. SSH_KEY="ssh/id_ed25519")
3. Заменить HOST_PATH, подставив туда путь с удаленного сервера (В формате для копирования с помощью scp)
4. Развернуть с помощью docker compose

```bash
docker compose up -d --build
```
